# SQL Sales & Customer Analytics Project

## Overview
This project analyzes sales performance, customer value, and customer retention using SQL.

## Business Problems
- Daily & monthly revenue analysis
- Month-over-month growth
- Top customers and city-wise revenue
- New vs repeat customer analysis

## Skills Used
- SQL (MySQL)
- CTEs
- Window Functions
- Aggregations & Joins
